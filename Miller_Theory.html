Вероятностные тесты простоты

Тест Миллера-Рабина

Тест Миллера—Рабина является на сегодняшний день вероятностным тестом, который чаще всего используется для проверки чисел на простоту.

Он основан на следующем свойстве простых чисел: если для нечетного простого p числа p - 1 записано в виде p - 1 = 2kd, где k - целое неотрицательное число.

Число d, в свою очередь, нечетное число. То для любого целого а, взаимно простого с p, имеет место либо сравнение ad ≡ 1(mod p), либо сравнение a2rd ≡ -1(mod p) для некоторого r ∈ [0, k-1].

Тест Миллера—Рабина использует указанные соотношения: для дан­ного нечетного n, записав число n — 1 в виде n — 1 = 2kd, мы произвольным образом выбираем натуральное а, 2 < а < n — 2 и проверяем сравнение ad ≡ 1 (mod n).

Если оно выполнено, то мы предполагаем, что число n — простое. В противном случае мы проверяем выполнимость одного из срав­нений a2rd ≡ -1(mod n) для r &isin [0, n — 1]. Если такое сравнение найдено, то мы предполагаем, что число n — простое. Если нет, то можем быть уверены, что n — составное.

Алгоритм теста Миллера-Рабина.

Вход: Нечетное целое число n ≥ 5.

Выход:"Число n составное" или "Число n, вероятно, простое".

Шаг 1. Представить n - 1 в виде n - 1 = 2sr, где число r нечетное.

Шаг 2. Выбрать случайное число a, 2 < a < n - 2.

Шаг 3. Вычислить y ≡ ar(mod n)

Шаг 4. При y ≠ 1 и y ≠ n - 1 выполнить:

4.1. Положить j = 1.

4.2. Если j ≠ s - 1 и y ≠ n - 1, то

4.2.1. Положить y ≡ y2(mod n)

При y = 1 результат: "Число n составное."

4.2.2. Положить j = j + 1.

При y = 1 результат: "Число n составное."

Шаг 5. Результат: "Число n, вероятно, простое".

Сложность алгоритма Миллера-Рабина равна O(log3n).
