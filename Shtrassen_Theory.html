Вероятностные тесты простоты

Тест Соловея-Штрассена

Тест Соловея—Штрассена — еще один вероятностный тест для опре­деления простоты данного натурального числа.

По критерию Эйлера, для любого нечетного простого p b любого a ∈ [1, p-1] имеет место сравнение

(a/p) ≡ a(p-1)/2(mod p)

Где (a/p) - символ Лежандра. Символ Якоби (a/n) является обобщением символа Лежандра, обладает аналогичными свойствами и может быть вычислен для любого нечетного n.

При использовании приведенных ниже свойств символ Якоби может быть вычислен за время O(log2n).

1. Если a ≡ b(mod n), то (a/n) = (b/n).

2. (ab/n) = (a/n)(b/n).

3. (a2/n) = 1; В частности, (1/n) = 1.

4. (-1/n) = (-1)(n-1)/2, то есть (-1/n) = 1, если n ≡ 1(mod 4), и (-1/n) = -1, если n ≡ -1(mod 4)

5. (2/n) = (-1) (n2-1)/8, то есть (2/n) = 1, если n ≡ ±1(mod 8), и (2/n) = -1, n ≡ ±3(mod 8).

6. Для различных нечетных натуральных чисел n и m имеет место равенство (n/m) = (-1)(n-1)/2 * (m-1)/2(n/m) (квадратичный закон взаимности).

Алгоритм теста Соловея-Штрассена.

Вход: Нечетное целое число n ≥ 5.

Выход:"Число n составное" или "Число n, вероятно, простое".

Шаг 1. Выбрать случайное число а, 2 < a < n - 2.

Шаг 2. Вычислить r = a (n-1)/2(mod n)

При r ≠ 1 и r ≠ n-1 результат: "Число n составное".

Шаг 3. Вычислить символ Якоби s = (a/n)

При r ≡ s(mod n) результат: "Число n, вероятно, простое."

В противном случае результат: "Число n составное."

Сложность теста Соловея-Штрассена определяется сложностью вычисления символа Якоби и равна O(log3n).
